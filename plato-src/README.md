# Plato Source Files

Canonical location for the Plato standard library source used to generate geometry and numeric types.

This is not a compiled project. The `.csproj` file exists only to make navigation and editing easier in the IDE.

When consumed from the [Ara 3D studio](https://github.com/ara3d/studio) monorepo, this folder lives at `submodules/Plato/plato-src`.
